# ☁️ AWS Cloud Practitioner Flashcards

Aplicación web interactiva para preparar la certificación **AWS Certified Cloud Practitioner (CLF-C02)** mediante tarjetas de estudio (Flashcards) y modo examen.

## 🚀 Abrir la aplicación

👉 https://manupp056.github.io/aws-flashcards/

---

# 📚 Características

- 🎴 Modo Flashcards
  - Estudia conceptos de AWS mediante tarjetas.
  - Haz clic sobre la tarjeta para ver la respuesta.
  - Navegación entre preguntas.
  - Pregunta aleatoria.
  - Barra de progreso.

- 📝 Modo Examen
  - Simulación de examen tipo test.
  - Respuestas de opción múltiple.
  - Corrección inmediata.
  - Preparación orientada al examen CLF-C02.

- 🗂️ Preguntas organizadas por categorías

Entre ellas:

- Monitoring
- Costos
- IAM
- EC2
- Almacenamiento
- Bases de Datos
- Redes
- Seguridad
- Arquitectura
- Serverless
- Migración
- Soporte
- Snow Family
- Shared Responsibility

---

# 🎯 Objetivo

Esta aplicación está diseñada para facilitar la preparación de la certificación oficial:

**AWS Certified Cloud Practitioner (CLF-C02)**

El contenido está organizado por temas para reforzar el aprendizaje progresivo y mejorar la retención mediante repetición activa.

Cada cambio enviado a la rama `main` se despliega automáticamente.

---

# 📖 Uso

1. Selecciona una categoría.
2. Lee la pregunta.
3. Haz clic en la tarjeta para descubrir la respuesta.
4. Continúa hasta completar todas las preguntas.
5. Cambia al **Modo Examen** para comprobar tus conocimientos.
